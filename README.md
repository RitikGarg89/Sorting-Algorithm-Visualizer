# AlgoVisualizer 🚀

A premium, highly interactive **Sorting Algorithm Visualizer** built with Vanilla JavaScript, HTML5, and CSS3. Designed with a sleek glassmorphism aesthetic and fully responsive across all devices.

![AlgoVisualizer Banner](/C:/Users/ritik/.gemini/antigravity/brain/cde76752-5955-4d0b-b6d2-0bd568836ee4/desktop_view_1772966374912.png)

## ✨ Features

- **Multiple Sorting Algorithms**: Visualize Bubble Sort, Selection Sort, Insertion Sort, Quick Sort, Merge Sort, Heap Sort, and Shell Sort.
- **Dynamic Visualizations**: Real-time color-coded **glowing animations** for comparisons (Red), swaps (Yellow), and sorted elements (Green).
- **Code Insights**: View the actual implementation of each algorithm in multiple languages (JavaScript, Python, C++, C, Java) directly within the app.
- **Interactive Controls**: 
  - **Array Size**: Adjust the number of elements to sort.
  - **Sorting Speed**: Control the animation speed from slow to lightning-fast.
  - **New Array**: Generate a fresh set of random data at any time.
- **Step-by-Step Summary**: After sorting, review a detailed log of every comparison and swap made by the algorithm.
- **Complexity Analysis**: Displays Time and Space complexity for the selected algorithm.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile with a fluid, stacked layout on smaller screens.

## 🎨 Design Aesthetic

- **Glassmorphism**: Modern UI using backdrop blurs and subtle borders.
- **Dynamic Backgrounds**: Animated radial gradients for a premium feel.
- **Micro-animations**: Smooth transitions and hover effects using CSS cubic-bezier.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties and complex media queries.
- **JavaScript**: ES6+ async/await logic for non-blocking animations.
- **Google Fonts**: 'Outfit' for modern typography.

## 🚀 Getting Started

Simply open `index.html` in your browser, or run a local server:

```bash
# Using npx
npx serve .
```
